Keyscores Embed Example

This is a plain HTML document, with a Keyscores embed code.

Single Chart

The embed URI structure is the following: 

          <iframe
            src="[URL]/[Token]/?[Arguments]"
            width="100%" height="220" frameborder="0" > </iframe>

This is the URL for the iFrame embed above: 

          <iframe
            src="https://demo.keyscores.com/media/default/embed/709579b6fb51ba2d1d87a87f5cd3169718f62ab3/?h=150&measure=TRANSFER"
            width="100%" height="220" frameborder="0" > </iframe>
You can see the parts of the URI are:

  • URL - the application location, the media demo is demo.keyscores.com/media/default/embed
  • Token - 709579b6fb51ba2d1d87a87f5cd3169718f62ab3 - this is the single-sign-on token of a user which can be used for embeddable publishing.
  • h - height of the chart, increments of 50 from 100 to 450 will be acceptable. This is not the same as the iFrame's borders.
  • measure=CODE; - the code for the measure displayed, you can also use measure=CODEA&measure=CODEB to produce multiple series
